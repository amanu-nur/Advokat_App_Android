# Advokat Aplication With React Native

This application is built using [React Native](https://reactnative.dev/) & [Firebase](https://firebase.google.com/), this application is intended for work at BNLF - Tulungagung - East Java - Indonesia.

## Installation

Use [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) or [yarn](https://yarnpkg.com/) to support installing/running this app.

```bash
#Clone This Repository
git clone https://github.com/amanu-nur/Advokat_App_Android.git

#Install Package With npm or yarn

#npm
npm install

#yarn
yarn install

#Running Aplications

#npm
npm run start     #=> To run metro server
npm run android   #=> To run in Android
npm run ios.      #=> To run in Ios

#yarn
yarn start     #=> To run metro server
yarn android   #=> To run in Android
yarn ios.      #=> To run in Ios
```
